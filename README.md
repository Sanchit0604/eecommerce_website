ShopZone — E-Commerce Website (Experiment 7)
A static e-commerce website with product listing, shopping cart, and checkout simulation.
How to Run
Open `index.html` directly in any web browser — no server required.
Structure
```
ecommerce_website/
├── index.html     # Main HTML page
├── style.css      # Stylesheet
├── script.js      # Cart logic & interactivity
└── README.md
```
Features
6 product cards with name, price, description
Add to Cart with quantity +/− controls
Dynamic subtotal & grand total calculation
Free shipping on orders above ₹2000
Checkout form with validation
Order confirmation with generated Order ID
